ma:action-msgs

A message, warning, or error message. This object is used by Media Album to communicate messages to the user.
Attributes
key A key representing a type of message.
Elements
msg A message string.

Definition

<xsd:complexType name="action-msgs"> <xsd:sequence> <xsd:element name="msg" type="xsd:string" maxOccurs="unbounded"/> </xsd:sequence> <xsd:attribute name="key" type="xsd:string" use="optional"/> </xsd:complexType>

Example instance

<action-messages key="org.apache.struts.action.GLOBAL_MESSAGE"> <msg>The collection 'Good Times' has been created.</msg> </action-messages>
