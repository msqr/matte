SearchResultAttributes

Set of attributes for search result data.
Attributes
is-partial-results If true then the search results returned are not all the matched results. Usually this means the search results returned are a subset of the results, to support paging across the results.
returned-results The number of search results returned.
search-time The number of milliseconds the search took to execute.
starting-offset A page offset for the returned results.
total-results The total number of results that matched the search query. Could be larger than the @returnedResults result if the search returned only a page of the results.
