ma:browse-data

This object is the root XML object used for browsing media collections.
Extends ma:abstract-data
Attributes
browseName The type of browsing currently being performed. This changes as different browse modes are selected, e.g. by date, by popularity, etc.
displayAlbum The @albumId of the ma:album to display.
displayItem The @itemId of the ma:media-item to display.
displayPage The page number, starting at 0 (zero), to display.
key The @anonymousKey of the ma:user media collection being browsed.
sort The sort mode applied to the browsing.
totalAlbums The total number of albums being browsed.
totalItems The total number of media items being browsed.
totalPages The total number of pages being browsed.
Elements
settings The current view settings to use while rendering to HTML, in the form of a ma:media-album-settings object. Since users can specify their view settings at any time, this object is used to tell their current settings.
theme The current ma:theme being used.
album-crumb The bread crumb trail of albums the user selected to view the current album or item, as ma:album-crumb objects.
album The ma:album objects to display.
