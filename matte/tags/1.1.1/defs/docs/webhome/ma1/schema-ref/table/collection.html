ma:collection

A collection is a collection of media items, used to help organize large collections of media items. Meida items always reside within one and only one collection.
Attributes
collectionId A unique number among all collection objects.
owner The @userId of the ma:user that owns this collection.
Elements
path A relative path where this collection resides on the server.
name The display name of the collection.
comment A description of the collection.
scandate The last time the collection was scanned, which approximates the last time a media item was added to the collection.
item A list of ma:media-item objects that are within the collection. This is not populated for album slideshows.

Definition

<xsd:complexType name="collection"> <xsd:sequence> <xsd:element name="path" type="xsd:string"/> <xsd:element name="name" type="xsd:string" minOccurs="0"/> <xsd:element name="comment" type="xsd:string" minOccurs="0"/> <xsd:element name="scandate" type="xsd:dateTime" minOccurs="0"/> <xsd:element name="item" type="media-item" minOccurs="0" maxOccurs="unbounded"/> </xsd:sequence> <xsd:attribute name="id" type="xsd:integer" use="optional"/> <xsd:attribute name="collectionId" type="xsd:integer"/> <xsd:attribute name="owner" type="xsd:integer"/> </xsd:complexType>

Example instance

<collection collectionId="1" owner="7"> <path>test/1079127525550/</path> <name>Default</name> <scandate>2004-03-12T14:15:39.581-08:00</scandate> <item creationDate="2002-12-11T21:18:54.000-08:00" fileSize="740951" collection="1" height="1600" hits="7" itemId="1" tzName="GMT-08:00" width="1200"> <path>IMG_0900.jpg</path> <mime>image/jpeg</mime> <name>Yee haw</name> <meta>|aCanon|bCanon PowerShot S330|c6|d6|e2|f89</meta> </item> </collection>
