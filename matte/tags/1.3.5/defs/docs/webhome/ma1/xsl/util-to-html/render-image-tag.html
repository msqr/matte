render-image-tag

Type Named
Parameters
item The ma:media-item to render the <img> element for.
size The size constant to pass to Media Server.
compress The compress constant to pass to Media Server.
original optional If set to anything, then render the Media Server URL as an original media item request.
onload optional If set to anything, then add an onload attribute to the <img> element with this as the attribute value.
id optional If set to anything, then add an id attribute to the <img> element with this as the attribute value.
max-width optional The maximum width to use for the image. If not provided then the maximum width is determined from the size parameter value. For performance it is better to store this value as a global variable and pass as a parameter here.
max-height optional The maximum height to use for the image. If not provided then the maximum height is determined from the size parameter value. For performance it is better to store this value as a global variable and pass as a parameter here.
Result HTML <img> element
Requires
  1. render-media-server-url
This template renders an HTML <img> element for a Media Server image object. You must supply the ma:media-item and desired size and compression level as parameters. This template calculates the width and height attribute values for the <img> element based on the size parameter value.

Example usage

<xsl:template match="ma:item" mode="thumbnail"> <td> <a href="javascript:parent.addImageToAlbum({position()},collectionData)"> <xsl:call-template name="render-image-tag"> <xsl:with-param name="max-width" select="$thumbMaxWidth"/> <xsl:with-param name="max-height" select="$thumbMaxHeight"/> <xsl:with-param name="item" select="."/> <xsl:with-param name="compress" select="$thumbCompress"/> <xsl:with-param name="size" select="$thumbSize"/> </xsl:call-template> </a> </td> </xsl:template>

Example output

<img height="90" width="68" src="/ma/MediaServer?id=1&s=tnormal&c=normal" alt="Yee haw">
