ma:media-album-settings

Defines the media view settings to be applied to the current view. This includes the size and compression settings for thumbnail and fullsize media.
Attributes
mediaServer The type of media server being used, in the form of a ma:media-server-type object. Currently the only supported option is internal, but this was designed to facilitate alternate external services such as Apache modules.
Elements
thumbnail The thumbnail ma:image-spec setting.
single The full size ma:image-spec setting.
Referenced By
  1. ma:browse-data
  2. ma:media-album-data

Definition

<xsd:complexType name="media-album-settings"> <xsd:sequence> <xsd:element name="thumbnail" type="ma:image-spec"/> <xsd:element name="single" type="ma:image-spec"/> </xsd:sequence> <xsd:attribute name="mediaServer" type="media-server-type" use="optional" default="internal"/> </xsd:complexType>

Example instance

<settings mediaServer="internal"> <thumbnail compress="normal" height="90" size="normal" width="120"/> <single compress="normal" height="480" size="normal" width="640"/> </settings>
