ma:user

A user object.
Attributes
anonymousKey A key unique among all user objects, used in browse URLs.
singleSize The user's preferred media size for full-sized media. See the media size definition.
singleCompress The user's preferred media compression setting for full-sized media. See the media compression definition.
thumbSize The user's preferred media size for thumbnail-sized media. See the media size definition.
thumbCompress The user's preferred media compression setting for thumbnail-sized media. See the media compression definition.
tz The user's default time zone offset, in number of milliseconds offset from GMT.
tzName A display name for the user's @tz setting. This value is set automatically by Media Album.
userId A number unique among all user objects.
Elements
username The logon username, which is unique among all user objects.
name The first and last names of the user.
email The email address of the user.
password The user's password, encrypted as a SHA-1 digest.
permissions The user permissions assigned to the user, in the form of a ma:permissions object.

Definition

<xsd:complexType name="user"> <xsd:sequence> <xsd:element name="username" type="xsd:string"/> <xsd:element name="name" type="xsd:string"/> <xsd:element name="email" type="xsd:string"/> <xsd:element name="password" type="xsd:string"/> <xsd:element name="permissions" type="permissions" minOccurs="0"/> </xsd:sequence> <xsd:attribute name="id" type="xsd:integer" use="optional"/> <xsd:attribute name="userId" type="xsd:integer"/> <xsd:attribute name="thumbSize" type="xsd:string"/> <xsd:attribute name="thumbCompress" type="xsd:string"/> <xsd:attribute name="singleSize" type="xsd:string"/> <xsd:attribute name="singleCompress" type="xsd:string"/> <xsd:attribute name="anonymousKey" type="xsd:string" use="optional"/> <xsd:attribute name="tz" type="xsd:integer" use="optional"/> <xsd:attribute name="tzName" type="xsd:string" use="optional"/> </xsd:complexType>
