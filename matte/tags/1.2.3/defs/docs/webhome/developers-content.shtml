There are many ways to contribute to the development of Matte, including Java, XSLT, HTML/CSS, documentation, and localization.

If you're interested in helping out in any way, please email msqr@users.sourceforge.net. If you've developed a theme you'd like to contibute to be part of the core distribution of Matte, email msqr@users.sourceforge.net.

Here are some links to some Matte developer resources.

Matte JavaDocs
For developers wishing to extend or customize Matte, the current JavaDocs are available here.
Theme HOWTO
The Theme HOWTO provides information about developing a Matte theme to customize the look and feel of Matte.
Extending media support
The Extending media support document provides information about developing your own media type handler for Matte.
Extending with plugins
  • The Extending with plugins document provides information about developing Matte's plugin support.
  • The Extending with browse mode plugins document provides information about developing "browse mode" plugins.
