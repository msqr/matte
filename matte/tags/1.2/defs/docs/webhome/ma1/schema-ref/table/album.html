ma:album

The album object that represents a collection of media items. An album can contain other albums within it (nested albums). Albums can be defined by users or by Media Album in the case of virtual albums to represent virtual browse modes (for example, year albums for browsing by date).
Attributes
albumDate A date to display as the real album date instead of the @creationDate. This allows users to specify a date for the album that differs from the album creation date.
albumId A number unique among all album objects.
allowAnonymous If true then this album allows anonymous browsing.
allowOriginal If true then this album allows users to download the original media items within it.
anonymousKey A unique value among all album objects that is used to view the album's slideshow. This is the key value you see in Media Album slideshow URLs.
creationDate The date the album was created.
modificationDate The date the album was last modified.
owner The @userId of the ma:user that owns this album.
parentId The @albumId of the album this album is nested within.
posterId The @itemId of the ma:media-item that should be used as the poster for this album. The poster is the media item to display as the representitive for the album.
sortMode A flag indicating the sort mode applied to this album.
themeId The @themeId of the ma:album-theme that should be used to render this album.
Elements
name The display name of this album.
comment A description or caption for the album.
item A list of ma:media-item objects that belong in this album.
permissions A list of ma:album-permissions objects that represent friend and group permissions for this album.
album A list of nested albums.

Definition
