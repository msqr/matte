ma:abstract-data

This abstract class is the base class for both the ma:media-album-data and ma:browse-data classes.
Attributes
admin Either true or false. If true then the current user is a super admin user.
Elements
user The currently logged in ma:user. If the current user is not logged in (i.e. anonymous) then this element will not be present.
action-messages Any errors or messages, represented by ma:action-msgs objects.
Referenced By
  1. ma:browse-data
  2. ma:media-album-data

Definition

<xsd:complexType name="abstract-data" abstract="true"> <xsd:sequence> <xsd:element name="user" type="ma:user" minOccurs="0"/> <xsd:element name="action-messages" type="ma:action-msgs" minOccurs="0" maxOccurs="unbounded"/> </xsd:sequence> <xsd:attribute name="admin" type="xsd:boolean"/> </xsd:complexType>
