Developers

Developers, developers, developers!

There are many ways to contribute to the development of Media Album, including Java, XSLT, HTML/CSS, documentation, and localization.

If you're interested in helping out, please email msqr@users.sourceforge.net. Here are some links to some Media Album developer resources.

Theme HOWTO
The Theme HOWTO provides information about developing a XSLT + CSS theme to customize the look and feel of Media Album.
Developer Jumpstart
The Developer Jumpstart describes how to get your own development environment set up to run Media Album within Eclipse.
XML Schema Object Reference
Check out the Media Album XML Schema Reference for information on the XML objects used by Media Album.
XSLT Template Reference
Check out the Media Album XSLT Template Reference for information on the XSLT templates available for theme development.
