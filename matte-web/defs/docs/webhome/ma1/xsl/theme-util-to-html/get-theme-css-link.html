get-css-link

Type Named
Parameters
theme The ma:album-theme to render the URL for.
css The CSS resource path to use.
ua-type The browser type retured from get-browser-type.
ua-version The browser version retured from get-browser-version.
ua-platform The browser platform retured from get-browser-platform.
Result HTML <link> element
Requires
  1. get-css-link
This template is a shortcut for calling get-css-link for the default theme CSS resource.

Example usage

<xsl:variable name="ua-type"> <xsl:call-template name="get-browser-type"> <xsl:with-param name="ua" select="$user-agent"/> </xsl:call-template> </xsl:variable> <xsl:variable name="ua-platform"> <xsl:call-template name="get-browser-platform"> <xsl:with-param name="ua" select="$user-agent"/> </xsl:call-template> </xsl:variable> <xsl:variable name="ua-version"> <xsl:call-template name="get-browser-version"> <xsl:with-param name="ua" select="$user-agent"/> </xsl:call-template> </xsl:variable> <xsl:call-template name="get-theme-css-link"> <xsl:with-param name="theme" select="ma:theme"/> <xsl:with-param name="ua-type" select="$ua-type"/> <xsl:with-param name="ua-version" select="$ua-version"/> <xsl:with-param name="ua-platform" select="$ua-platform"/> </xsl:call-template>

Example output

/ma/theme/1/1029809801/album.css

Source

<xsl:template name="get-theme-css-link"> <xsl:param name="theme"/> <xsl:param name="ua-type"/> <xsl:param name="ua-version"/> <xsl:param name="ua-platform"/> <xsl:call-template name="get-css-link"> <xsl:with-param name="theme" select="$theme"/> <xsl:with-param name="css" select="$theme/ma:css"/> <xsl:with-param name="ua-type" select="$ua-type"/> <xsl:with-param name="ua-version" select="$ua-version"/> <xsl:with-param name="ua-platform" select="$ua-platform"/> </xsl:call-template> </xsl:template>
