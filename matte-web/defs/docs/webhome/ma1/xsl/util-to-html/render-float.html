render-float

Type Named
Parameters
num The floating point number to round.
places The maximum number of decimal places to display.
Result float

Round a floating point number to a specified number of decimal places. This is useful for rounding a number to a specific precision.

Example usage

<xsl:call-template name="render-float"> <xsl:with-param name="num" select="2506 div 1024"/> <xsl:with-param name="places" select="'2'"/> </xsl:call-template>
