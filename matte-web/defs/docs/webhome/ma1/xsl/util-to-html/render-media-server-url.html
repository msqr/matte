render-media-server-url

Type Named
Parameters
item The ma:media-item to render the Media Server URL for.
size The size constant to pass to Media Server.
compress The compress constant to pass to Media Server.
original optional If set to anything, then render the Media Server URL as an original media item request.
Globals
  1. web-context
Result string (URL)
Referenced By
  1. render-image-tag
Renders a string Media Server URL for a media item.

Example usage

<img width="{$rWidth}" height="{$rHeight}"> <xsl:attribute name="src"> <xsl:call-template name="render-media-server-url"> <xsl:with-param name="item" select="$item"/> <xsl:with-param name="original" select="false()"/> <xsl:with-param name="size" select="'normal'"/> <xsl:with-param name="compress" select="'normal'"/> </xsl:call-template> </xsl:attribute> </img>

Example output

/ma/MediaServer?id=1&s=tnormal&c=normal

Source

<xsl:template name="render-media-server-url"> <xsl:param name="item"/> <xsl:param name="compress"/> <xsl:param name="size"/> <xsl:param name="original"/> <xsl:value-of select="$web-context"/> <xsl:text>/MediaServer?id=</xsl:text> <xsl:value-of select="$item/@itemId"/> <xsl:choose> <xsl:when test="$original"> <xsl:text>&amp;original=true</xsl:text> </xsl:when> <xsl:otherwise> <xsl:text>&amp;s=</xsl:text> <xsl:value-of select="$size"/> <xsl:text>&amp;c=</xsl:text> <xsl:value-of select="$compress"/> </xsl:otherwise> </xsl:choose> </xsl:template>
