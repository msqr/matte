render-browse-dl

Type Matching
Matches ma:user or ma:friend
Result HTML fragment
This template renders an HTML <dl> list of browsing links for a user. These links are displayed on the My Settings page as well as the Friends & Groups page when a friend is selected.

Example usage

<xsl:template match="ma:user"> <xsl:apply-templates select="." mode="render-browse-dl"/> </xsl:template>

Example output
