ma:album-crumb

A browse bread crumb which is used for displaying the browse heirarchy a user is at given a nested album or album item.
Attributes
album The @albumId of the ma:album crumb.
page The page number the album resides on.
Elements
name The name of the ma:album.
Referenced By
  1. ma:browse-data

Definition

<xsd:complexType name="album-crumb"> <xsd:sequence> <xsd:element name="name" type="xsd:string"/> </xsd:sequence> <xsd:attribute name="page" type="xsd:integer" use="required"/> <xsd:attribute name="album" type="xsd:integer" use="required"/> </xsd:complexType>

Example instance

<album-crumb album="2002" page="0"> <name>2002</name> </album-crumb>
