ma:image-spec

Encapsulates the view options for media items. Currently this includes size and compression values. These are in the form of keywords, and if an unsupported keyword is found Media Album uses the normal keyword by default.
Compression
Keyword Quality
none 100
normal 75
medium 50
high 30
Note the compression values listed here are approximate quality values, where 0 is the worst quality (highest compression) and 100 is the highest quality (lowest compression).
 Size:
Keyword Dimensions (w x h)
tsmall 80 x 60
tnormal 120 x 90
tbig 160 x 120
thuge 240 x 180
small 320 x 240
normal 640 x 480
medium 800 x 600
big 1024 x 768
huge 1600 x 1200
The dimension values listed here are in pixels.
Attributes
compress A compression keyword.
size A size keyword.

Definition

<xsd:complexType name="image-spec"> <xsd:attributeGroup ref="WidthHeightAttributes"/> <xsd:attribute name="compress" type="xsd:string" use="optional" default="normal"/> <xsd:attribute name="size" type="xsd:string" use="optional" default="normal"/> </xsd:complexType>

Example instance

<thumbnail compress="normal" height="90" size="normal" width="120"/>
