ma:album-theme

The theme object represents a Media Album theme that can be used to view album slideshows or browse media collections with. Themes are used to provide alternate output styles, and users are able to upload their own themes and apply them to albums in addition to their default browse experience.
In general a theme consists of an XSLT fragment which is responsible for rendering Media Album XML data into HTML, a CSS document for applying CSS styles to the output HTML, an icon image that displays while browsing themes within Media Album, and a larger preview image that is used to display a preview of what the output HTML of the theme looks like.
Attributes
author The first and last name of the theme's author.
authorEmail The email address of the author.
creationDate The date the theme was created.
global If true then the theme can be used by all Media Album users. Otherwise it can only be used by the theme's owner.
modificationDate The date the theme was last modified.
owner The @userId of the ma:user that owns the theme.
themeId A number unique among all theme objects.
Elements
name The theme's name.
comment A description of the theme.
base-dir A relative directory that serves as the base directory for theme files. This value is assigned automatically by Media Album when creating the theme.
icon The name of the thumbnail-sized icon image file.
preview The name of the preview image file.
xsl The name of the theme XSLT fragment file.
css The name of the CSS file.

Definition

<xsd:complexType name="album-theme"> <xsd:sequence> <xsd:element name="name" type="xsd:string"/> <xsd:element name="comment" type="xsd:string"/> <xsd:element name="base-dir" type="xsd:string"/> <xsd:element name="icon" type="xsd:string"/> <xsd:element name="preview" type="xsd:string"/> <xsd:element name="xsl" type="xsd:string" minOccurs="0"/> <xsd:element name="css" type="xsd:string"/> </xsd:sequence> <xsd:attribute name="themeId" type="xsd:integer"/> <xsd:attribute name="owner" type="xsd:integer" use="optional"/> <xsd:attribute name="global" type="xsd:boolean" default="false"/> <xsd:attribute name="author" type="xsd:string" use="optional"/> <xsd:attribute name="authorEmail" type="xsd:string" use="optional"/> <xsd:attribute name="creationDate" type="xsd:dateTime"/> <xsd:attribute name="modificationDate" type="xsd:dateTime" use="optional"/> </xsd:complexType>

Example instance

<theme author="Matt Magoffin" authorEmail="spamsqr@msqr.us" creationDate="2004-03-12T12:57:16.161" global="true" owner="0" themeId="-1"> <name>Default Theme</name> <comment>This is the default theme.</comment> <base-dir>inc/standard/default/</base-dir> <icon>thumbnail.png</icon> <preview>preview.png</preview> <xsl>album.xsl</xsl> <css>album.css</css> </theme>
