ma:media-item

This object represents a single media item (picture, movie, etc.) in the Media Album application. A media item belongs to one and only one ma:collection object, but can live in any number of ma:album objects.
Attributes
creationDate The date the item was created. If the item supports embedded metadata and this metadata provides a creation date, that date is used. Otherwise the creation date will be set to the date the media item was added to Media Album.
displayOrder A display ordering, used for media items within ma:album objects.
fileSize The natural size of this item, in bytes.
collection The @collectionId of the ma:collection object this item belongs to.
itemId A unique number among all media item objects.
height The natural height of this item, in pixels.
hits The number of times this item has been viewed, not including thumbnail-sized views.
tz A time zone offset, in number of milliseconds offset from GMT.
tzName A display representation of the @tz time zone offset value. This is automatically populated by Media Album.
width The natural width of this item, in pixels.
Elements
path A file path where this media album resides, relative from the file path of this item's collection.
mime The MIME type of this item.
name The display name.
comment A comment or caption.
meta The raw metadata extracted from the media item, encoded as a string. This value is usually not used as the metadata element provides an easier data format for this information.
metadata The metadata extracted from the media item, in the form of ma:media-item-metadata objects.
user-rating List of user ratings assigned to this media item, in the form of ma:item-rating objects.
user-comment List of user comments assigned to this media item, in the form of ma:item-comment objects.
