If you use iPhoto to organise your photos on a Mac, you might like to try the Matte iPhoto Export Plugin, which allows you to export entire albums directly from iPhoto into Matte.

You can download the plugin from here. See the Matte iPhoto Export Plugin page for more information.

In addition, there is an AppleScript included in the binary distribution of Matte for exporting an album from iPhoto. This script may work with older version of iPhoto not supported by the Matte iPhoto Export Plugin. See the Matte iPhoto Export AppleScript page for more information.
