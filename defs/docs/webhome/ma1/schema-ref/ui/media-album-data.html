ma:media-album-data

This object is the root XML object used for viewing album slideshows.
Extends ma:abstract-data
Attributes
displayAlbum The @albumId of the ma:album to display. When an album has nested albums, this is the ID of the album to render.
displaySource The @collectionId of the ma:collection to display. Generally this is not used in theme development.
Elements
settings The current view settings to use while rendering to HTML, in the form of a ma:media-album-settings object. Since users can specify their view settings at any time, this object is used to tell their current settings.
theme The current ma:theme being used.
collection The collections the owner of this album data has, in the form of ma:collection objects.
album The ma:album objects to display. Generally for album slideshows there will be only one root album and possibly nested albums within that.
group The ma:group objects that the owner of the album data belongs to. This is not populated for album slideshows.
friend The ma:friend objects that the owner of the album data has. This is not populated for album slideshows.

Definition

<xsd:complexType name="media-album-data"> <xsd:complexContent> <xsd:extension base="abstract-data"> <xsd:sequence> <xsd:element name="settings" type="media-album-settings"/> <xsd:element name="theme" type="ma:album-theme" minOccurs="0"/> <xsd:element name="collection" type="ma:collection" minOccurs="0" maxOccurs="unbounded"/> <xsd:element name="album" type="ma:album" minOccurs="0" maxOccurs="unbounded"/> <xsd:element name="group" type="ma:group" minOccurs="0" maxOccurs="unbounded"/> <xsd:element name="friend" type="ma:user" minOccurs="0" maxOccurs="unbounded"/> </xsd:sequence> <xsd:attribute name="displayAlbum" type="xsd:integer" use="optional"/> <xsd:attribute name="displaySource" type="xsd:integer" use="optional"/> </xsd:extension> </xsd:complexContent> </xsd:complexType>
