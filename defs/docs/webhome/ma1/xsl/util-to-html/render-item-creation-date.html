render-item-creation-date

Type Named
Parameters
item The ma:media-item to render the creation date for.
user The ma:user to render the creation date for.
Result string
Requires
  1. render-timezone
  2. date:format-date
Renders the creation date of a media item based on the @creationDate attribute of the media item and the @tz attribute of the media item and of a user. The time zone will be appended to the date only if the time zone of the item is not the same as that of the user.

Example usage

<xsl:template match="ma:item"> <xsl:text>Created: </xsl:text> <xsl:call-template name="render-item-creation-date"> <xsl:with-param name="item" select="." /> <xsl:with-param name="user" select="/ma:media-album-data/ma:user"/> </xsl:call-template> </xsl:template>

Example output

26 Dec 2002, 5:15:28 PM (GMT+12:00)

Source
