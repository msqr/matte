render-file-size

Type Named
Parameters
size The length of the file, in bytes.
Result text
Requires
  1. render-float
Render the length of a file as mebibytes, kibibytes, or bytes with as most 2 decimal places.

Example usage

<xsl:call-template name="render-file-size"> <xsl:with-param name="size" select="19283754"/> </xsl:call-template>

Example output

18.39 MB

Source

<xsl:template name="render-file-size"> <xsl:param name="size"/> <xsl:choose> <xsl:when test="$size &gt; 1048576"> <xsl:call-template name="render-float"> <xsl:with-param name="num" select="$size div 1048576"/> <xsl:with-param name="places" select="'2'"/> </xsl:call-template> <xsl:text> MB</xsl:text> </xsl:when> <xsl:when test="$size &gt; 1024"> <xsl:call-template name="render-float"> <xsl:with-param name="num" select="$size div 1024"/> <xsl:with-param name="places" select="'2'"/> </xsl:call-template> <xsl:text> KB</xsl:text> </xsl:when> <xsl:otherwise> <xsl:value-of select="$size"/> <xsl:text> bytes</xsl:text> </xsl:otherwise> </xsl:choose> </xsl:template>
