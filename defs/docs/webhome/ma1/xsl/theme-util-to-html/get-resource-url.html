ger-resource-url

Type Named
Parameters
theme The ma:album-theme to render the URL for.
resource The relative path to the resource file.
Globals
  1. web-context
Result string (URL)
Referenced By
  1. get-css-link
Get a theme resource URL, i.e. a CSS, JavaScript, image, or any other supported theme file.

Example usage

<xsl:call-template name="get-resource-url"> <xsl:with-param name="theme" select="$theme"/> <xsl:with-param name="resource" select='browse.css'/> </xsl:call-template>

Example output

/ma/theme/1/1029809801/browse.css

Source

<xsl:template name="get-resource-url"> <xsl:param name="theme"/> <xsl:param name="resource"/> <xsl:value-of select="$web-context"/> <xsl:text>/theme/</xsl:text> <xsl:value-of select="$theme/ma:base-dir"/> <xsl:value-of select="$resource"/> </xsl:template>
