Matte: iPhoto Export Plugin

If you have a Mac and use iPhoto to organize your photos into albums, you can download and install the Matte iPhoto Export Plugin which will allow you to export albums easily into Matte, preserving your photo names, comments, keywords and ratings.

The iPhoto Export Plugin is currently a very early release. At present direct export of albums into Matte is not supported. Instead it will allow you to export an album as a zip archive that includes all your iPhoto metadata such as names, comments, keywords, and ratings. You can then import that zip archive into Matte using Matte's normal "Add" feature. The eventual goal for this plugin is to export albums directly from iPhoto into Matte.

Requirements

The plugin requires the following:

OS X
This plugin has been tested on OS X 10.5 on PPC G5 and G4 computers. It should work on OS X 10.4 but this has not been tested. It may or may not work on other versions of OS X.
iPhoto
The plugin has been tested only with iPhoto 7. It may or may not work on other versions of iPhoto.

Installing the plugin

Once you've downloaded the Matte iPhoto Export Plugin disk image, mount the disk image by double-clicking on it. Then run the installer application that comes in the mounted disk image.

Performing the export from iPhoto

Follow these steps to export an album from iPhoto:

Select album to export in iPhoto

Click on the album you want to export in iPhoto. This can be any album, including "smart" albums. Note that nested albums will not be exported.

Selecting album to export in iPhoto.

Start export

Choose to the File > Export menu in iPhoto, and then the Matte tab in the Export Photos dialog that appears.

The Matte iPhoto Export dialog

Update the settings in the export window to your liking and then click the Export button.

Please note that not all fields are supported yet. In particular the URL, Username, Password, Collection, and Auto Album options can be ignored. If you want to export your original photos (usually this is the desired option), make sure the Export originals button is checked.

The export plugin will then prompt you for a folder to export the album to. After choosing the folder, it will export the album, all of the photos in the album, create a metadata.xml file, and then a zip archive comprised of the album photos and the metadata.xml file.

The export results.

The zip archive will be named after the album you've exported, and this is the file you should then upload into Matte.

Import the zip archive into Matte

Now you can import the zip archive created by the previous step into Matte. Log into Matte, and from the Add screen, select zip file you just created, and be sure to check the Auto Album checkbox so that Matte re-creates your albums for you.

Import the items into Matte.

After the import completes, you will have the album available in Matte, ready to be shared.
